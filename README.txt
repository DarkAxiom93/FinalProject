# Casino Royal Terminal
פרויקט משחקי קזינו מקיף בשפת C המשלב לוגיקה מתמטית וניהול זיכרון מאובטח.

## תכונות מרכזיות
* **משחקים:** בלאק-ג'ק, פוקר, רולטה, מכונת מזל והימורי ספורט.
* **אבטחה:** מנגנון Anti-Cheat מבוסס Checksum למניעת זיופי יתרה.
* **ניהול זיכרון:** קוד נקי מ-Memory Leaks (נבדק עם Dr. Memory).

## איך מריצים?
1. פתח את קובץ ה-`.sln` ב-Visual Studio.
2. בחר במצב **Release**.
3. לחץ על **Build Solution**.
4. הפעל את ה-`casino.exe` שנוצר בתיקיית ה-Release.

## טכנולוגיות
- שפת C בתקן ANSI.
- ניהול זיכרון דינמי.
- עבודה עם קבצים בינאריים לשימור נתונים.

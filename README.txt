🎲 Virtual Casino System - Advanced C Project
מערכת קזינו וירטואלית מתקדמת בשפת C

פרויקט גמר זה מציג סימולציית קזינו שלמה, מאובטחת ומודולרית. 
המערכת תוכננה תוך הקפדה על עקרונות הנדסת תוכנה מודרניים (Separation of Concerns), ניהול זיכרון קפדני, ממשק משתמש מתקדם במסוף (Terminal UI) ומנגנוני אבטחת מידע.

✨ מאפייני ליבה טכנולוגיים (Core Features)
ארכיטקטורה מודולרית (High Cohesion, Low Coupling): הקוד מפוצל למודולים ייעודיים. לוגיקת המשחקים מופרדת לחלוטין ממנוע התצוגה (Graphics), ממערכת המשתמשים (Accounts) ומתשתיות העזר (Utils).

מנגנון Anti-Cheat אקטיבי: מערכת השמירות משתמשת בקבצים בינאריים (.bin) ומחשבת "חתימה מתמטית" (Checksum) לכל שחקן. כל ניסיון של משתמש לשנות את קובץ השמירה שלו ידנית יוביל לזיהוי הרמאות ולאיפוס חשבונו.

מנוע קלפים גנרי (Shared Engine): ספרייה מרכזית (cards.c) המייצרת ומערבבת חפיסות קלפים באופן דינמי באמצעות הקצאות זיכרון, ומשמשת גם את הפוקר (חפיסה אחת) וגם את הבלאק-ג'ק (6 חפיסות).

אבטחת קלטים (Bulletproof Input): כל קלט מהמשתמש עובר דרך פונקציות טיהור (Sanitation) המנקות את חוצץ המקלדת ומונעות קריסות (Buffer Overflow / Segmentation Faults).

ניהול זיכרון בטוח: שימוש ב-safe_malloc לעטיפת הקצאות זיכרון עם טיפול בשגיאות, והקפדה על שחרור משאבים (free) בסוף כל סיבוב למניעת זליגות זיכרון (Memory Leaks).

🎰 מגוון המשחקים במערכת
Roulette: לוח רולטה מלא הכולל הימורים מורכבים (Corners, Splits, Dozens), מערכת ביטול הימורים (Undo) ושמירת היסטוריית סיבובים אחרונים.

VIP Blackjack: משחק בלאק-ג'ק מקצועי המשתמש ב-Shoe של 6 חפיסות, תומך בפיצול (Split) ובכפל הימור (Double Down), עם לוגיקת דילר חכמה (Hit on Soft 17).

Ultimate Texas Hold'em: פוקר נגד הדילר. כולל הימורי Blind, Ante, ואופציה להימור Trips (שמשלם לפי חוזק היד ללא קשר לדילר). אלגוריתם זיהוי ידיים חזק מזהה מ-High Card ועד Straight Flush.

Slot Machine: מכונת מזל עם אנימציית גלילה מותאמת אישית, אפשרות להימורים מהירים ברצף (ללא לחיצה על Enter) ומערכת Jackpots.

Winner Sportsbook: מערכת הימורי ספורט דינמית (טופס טוטו). מגרילה משחקים ייחודיים ויחסי זכייה (Odds) מתוך מאגר קבוצות, ומדמה תוצאות משחקי כדורגל.

📁 ארכיטקטורת הקבצים (Project Structure)
הפרויקט מחולק בצורה לוגית למודולים הבאים:

Core System (ליבת המערכת)

main.c - לולאת המשחק המרכזית, תפריט הראשי וטבלת השיאים (Leaderboard).

account.c / .h - ניהול חשבונות, מערכת הבנק המוגנת (Safe Bank), כתיבה/קריאה בינארית ומנגנון ה-Anti-Cheat.

utils.c / .h - פונקציות עזר, טיהור קלטים (get_safe_int), השהיות זמן ומאקרואים של צבעי ANSI.

graphics.c / .h - ריכוז כל ציורי ה-ASCII ומסכי הפתיחה להפרדת ה-UI מהלוגיקה.

Shared Libraries (ספריות משותפות)

cards.c / .h - מנוע ייצור, ניהול וערבוב (Fisher-Yates) של חפיסות קלפים.

Game Modules (מודולי המשחקים)

roulette.c, blackjack.c, poker.c, slots.c, football.c - קבצים המכילים אך ורק את הלוגיקה והחוקים של כל משחק.

House Bonus: הענקת כסף לקופת השחקן הנוכחי.
